# kuzukiri
シンプルな文分割ツール

## これは何？
複数文からなる日本語文書を文ごとに分割するためのPythonライブラリです。

## 特徴
* シンプルなルールによる分割
  * 完全なルールベースです。機械学習等は使っていません
  * ルールがシンプルなので入力から結果を予測可能
* Rustで書かれているため高速 

## インストール
### ソースコードからインストールする場合

```bash
pip install setuptools-rust
python setup.py install
```
